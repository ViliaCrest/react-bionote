# Bionote - Personal Portfolio Website

A comprehensive React-based personal portfolio website that demonstrates React fundamentals, component creation, modern web development practices, and clean UI design. This project combines multiple challenge activities into a complete bionote application showcasing the profiles of Vilia Crestene and Carla Primitiva.

## Challenge Activities

**Challenge Activity 1: Component Creation**
- Login and Sign Up components (Car Component)
- Reusable Bionote components for profile display
- Header and Footer components

**Challenge Activity 2: Component Styling**
- Custom CSS styling with modern design principles
- Clean color scheme implementation (White & #01143D)
- Responsive layout and interactive UI elements

**Challenge Activity 3: Component Integration**
- Complete application structure with routing logic
- Profile selection and navigation system
- Integrated Header and Footer components

## Final Output

Complete Bionote Homepage with authentication flow, profile selection, and detailed bio note pages.

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Running the Application

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or the port shown in the terminal).

### Building for Production

To create a production build:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```

## Project Structure

- `app.jsx` - Main application component with routing logic
- `Car.jsx` - Login/Signup component
- `Bionote.jsx` - Profile component for Vilia Crestene
- `BionotePair.jsx` - Profile component for Carla Primitiva
- `src/components/Header.jsx` - Header component with navigation
- `src/components/Footer.jsx` - Footer component
- `main.jsx` - Application entry point
- `index.css` - Global styles
- `index.html` - HTML template
- `image/` - Profile images folder

## Features

- User authentication (login/signup)
- Profile selection
- Bio note display with skills, interests, education, and projects
- Modern, clean UI design
- Responsive layout
- Interactive components with hover effects
